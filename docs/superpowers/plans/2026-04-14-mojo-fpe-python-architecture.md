# Mojo FPE Solver Refactoring: Match Python Architecture

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refactor Mojo FPE solver to match Python's domain-transformation architecture, fixing all numerical errors and making results match Python baseline.

**Architecture:** Python works in normalized [0,1] coordinates with Jacobian factors for coordinate transformation. Mojo currently works in physical coordinates, causing large numerical values and incorrect results. The refactoring introduces: (1) normalized coordinate system throughout, (2) separate grid_create for quadrature points (not knots!), (3) proper Jacobian factors in stiffness matrix, (4) proper delta_approx in physical space mapped to normalized space, (5) proper OSQP with D-scaling.

**Tech Stack:** Mojo v0.26.3 (Nightly), pixi, existing sparse/numerics modules

## Critical Architecture Differences (Python vs Current Mojo)

### Python Architecture (CORRECT, baseline):
1. **Knots** are in [0,1] normalized space (GenerateKnots generates normalized knots)
2. **Quadrature points** are generated by `grid_create()` → normalize → `compute_quad()` per knot interval
3. **Basis evaluation** happens at normalized quadrature points
4. **s_points (physical)** = normalized_points * (S_max - S_min) + S_min (only for delta_approx and option pricing)
5. **s_weights** include Jacobian factor: `s_weights *= (S_max - S_min)`
6. **Stiffness matrix** uses `/j` factors: k1=(-r+0.5*rho*eta)/j, k2=1/j, k3=0.5/j², k4=0.5*rho*eta/j
7. **delta_approx** uses physical s_points and v_points with bivariate_normal
8. **q_initial** uses D-scaling: D=diag(sqrt(M_diag)), solve in scaled space
9. **option_pricing** uses marginal distribution (integrate over v) with s_weights (already has Jacobian)

### Current Mojo Architecture (WRONG):
1. **Knots** are in physical space (GenerateKnots converts to physical at line 172-177)
2. **Quadrature points** are generated from physical knots (wrong!)
3. **Basis evaluation** happens at physical quadrature points (wrong!)
4. **No s_points/v_points distinction** - everything is in physical space
5. **s_weights** don't include Jacobian factor (we removed it!)
6. **Stiffness matrix** has no /j factors (we removed them!)
7. **delta_approx** uses physical coordinates (correct but wrong grid)
8. **q_initial** uses simple ProjectedGradient (no D-scaling, no mass constraint)
9. **No option pricing** with marginal distribution

This is the core change. Python has two separate grids:
1. **knots** (in [0,1] space) - control B-spline basis generation

```mojo
def _grid_create(mean: Float64, std_dev: Float64, bound: Tuple[Float64, Float64], num_insert: Int = 251, is_v: Bool = False) -> List[Float64]:
    var lb = bound[0]
    var ub = bound[1]
    var lb_interm = mean - 5.0 * std_dev
    var ub_interm = mean + 5.0 * std_dev
    var num_interm = num_insert * 30 // 100
    var right_trail = num_insert * 50 // 100
    var left_trail = num_insert * 20 // 100
    
    var x: List[Float64] = []
    # Coarse lower tail
    var lt = _linspace(lb, lb_interm, left_trail)
    for i in range(len(lt)): x.append(lt[i])
    # Approaching mean
    var am = _linspace(lb_interm, mean - std_dev / 5.0, num_interm // 3)
    for i in range(len(am)): x.append(am[i])
    # Dense near mean
    var dn = _linspace(mean - std_dev / 5.0, mean + std_dev / 5.0, num_interm // 3)
    for i in range(len(dn)): x.append(dn[i])
    # Ensure mean is included
    x.append(mean)
    # Leaving mean
    var lm = _linspace(mean + std_dev / 5.0, ub_interm, num_interm // 3)
    for i in range(len(lm)): x.append(lm[i])
    # Coarse upper tail
    var rt = _linspace(ub_interm, ub, right_trail)
    for i in range(len(rt)): x.append(rt[i])
    # Upper region
    var ur = _linspace(ub - 0.1, ub, num_insert // 10)
    for i in range(len(ur)): x.append(ur[i])
    # For v_points, add extra near v=0
    if is_v:
        var zr = _linspace(0.0, 0.01, num_insert * 20 // 100)
        for i in range(len(zr)): x.append(zr[i])
    
    return _sort_unique(x)^
```

```mojo
def _normalize_list(x: List[Float64], lo: Float64, hi: Float64) -> List[Float64]:
    var out: List[Float64] = []
    var scale = hi - lo
    if scale <= 0.0: scale = 1.0
    for i in range(len(x)):
        out.append((x[i] - lo) / scale)
    return out^

def _compute_quad(grid: List[Float64], gl_order: Int) -> Tuple[List[Float64], List[Float64]]:
    # Get unique knots from grid for interval detection
    var unique: List[Float64] = []
    if len(grid) > 0:
        unique.append(grid[0])
        for i in range(1, len(grid)):
            if grid[i] > unique[len(unique)-1] + 1e-9:
                unique.append(grid[i])
    
    var points: List[Float64] = []
    var weights: List[Float64] = []
    var gl = GaussLegendre(gl_order)
    var n_intervals = len(unique) - 1
    for i in range(n_intervals):
        var a = unique[i]
        var b = unique[i + 1]
        if b <= a: continue
        var half_span = 0.5 * (b - a)
        var mid = 0.5 * (a + b)
        for j in range(gl.order):
            points.append(half_span * gl.nodes[j] + mid)
            weights.append(half_span * gl.weights[j])
    return (points^, weights^)

def _sort_unique(mut x: List[Float64]) -> List[Float64]:
    for i in range(len(x)):
        for j in range(i + 1, len(x)):
            if x[i] > x[j]:
                var tmp = x[i]; x[i] = x[j]; x[j] = tmp
    var out: List[Float64] = []
    if len(x) > 0:
        out.append(x[0])
        for i in range(1, len(x)):
            if x[i] > out[len(out)-1] + 1e-12:
                out.append(x[i])
    return out^
```

### Task 4: Rewrite Initial Condition - Proper delta_approx + D-scaling OSQP

**Files:**
- Rewrite: `src/engines/fpe/initial_cond.mojo`
- Rewrite: `src/numerics/optim/osqp.mojo`

Python's `delta_approx()`:
- Uses `multivariate_normal(mean=[S0, V0], cov=...)` in physical space
- `s_range = self.s_points` (physical), `v_range = self.points_list[1]` (normalized [0,1])
- `v_stdev = sigma0 / (s_range.max() - s_range.min())`
- Evaluates PDF at meshgrid of (physical_s, normalized_v)

Python's `q_initial()`:
- Uses D-scaling: `D = diag(sqrt(M_diag))`, `Dinv = diag(1/sqrt(M_diag))`
- Scales: `M_scaled = Dinv @ M @ Dinv`, `galerkin_proj = Dinv @ (Phi^T @ W @ delta)`
- Solves OSQP: `min 0.5 ||M_scaled @ c - galerkin_proj||² + 1e-6||c||² s.t. c >= 0`
- Unscales: `result = Dinv @ c.value`
- Normalizes: `result /= m^T @ result`

- [ ] **Step 1: Rewrite _bivariate_gaussian to match Python's delta_approx**

```mojo
def _delta_approx(domain: FPEDomain, params: HestonParams, sigma0: Float64) -> List[Float64]:
    var s0 = params.S0
    var v0 = params.V0
    var s_range_span = domain.s_max - domain.s_min
    var v_stdev = sigma0 / s_range_span if s_range_span > 0.0 else sigma0
    var s_sigma = sigma0
    var v_sigma = v_stdev
    
    var norm = 1.0 / (2.0 * pi * s_sigma * v_sigma)
    var out: List[Float64] = []
    for i in range(len(domain.s_points_phys)):
        var s = domain.s_points_phys[i]
        var ds = (s - s0) / s_sigma
        for j in range(len(domain.v_points_phys)):
            var v = domain.v_points_phys[j]
            var dv = (v - v0) / v_sigma
            var val = norm * exp(-0.5 * (ds * ds + dv * dv))
            out.append(val)
    
    # Clip small values like Python
    for i in range(len(out)):
        if out[i] < 1e-6: out[i] = 0.0
        if out[i] < 0.0: out[i] = 0.0
    
    return out^
```

- [ ] **Step 2: Rewrite OSQP with D-scaling**

Add D-scaling to the OSQP solve. The key insight: Python solves in scaled space where M_scaled is well-conditioned.

```mojo
struct InitialCondition[B: Int]:
    def __init__(out self): pass

    def compute(self, domain: FPEDomain, params: HestonParams, sigma0: Float64 = 0.1) raises -> List[Float64]:
        var basis = domain.build_basis()
        var Phi = basis.eval_tensor(domain.s_points, domain.v_points)
        var delta = _delta_approx(domain, params, sigma0)
        
        # Build mass matrix
        var w = _build_weight_vector(domain)
        var M = _spT_diag_sp(Phi, w)
        
        # Galerkin projection: Phi^T @ W @ delta
        var galerkin_proj = _spT_vec(Phi, w, delta)
        
        # D-scaling
        var M_diag = _csr_diag(M)
        var Dinv_diag: List[Float64] = []
        for i in range(len(M_diag)):
            Dinv_diag.append(1.0 / sqrt(M_diag[i])) if M_diag[i] > 0.0 else Dinv_diag.append(0.0)
        
        # Scale galerkin_proj: Dinv @ galerkin_proj
        var galerkin_proj_scaled: List[Float64] = []
        for i in range(len(galerkin_proj)):
            galerkin_proj_scaled.append(Dinv_diag[i] * galerkin_proj[i])
        
        # Scale M: Dinv @ M @ Dinv (apply left and right)
        var M_scaled = _scale_matrix_left_right(M, Dinv_diag, Dinv_diag)
        
        # Solve NNLS: min 0.5 ||M_scaled @ c - galerkin_proj_scaled||² s.t. c >= 0
        var M_scaled_dense = _csr_to_dense(M_scaled)
        var pg = ProjectedGradient(max_iter=50000, tol=1e-8, step_size=-1.0)
        var c = pg.solve(M_scaled_dense, galerkin_proj_scaled)
        
        # Unscale: result = Dinv @ c
        var result: List[Float64] = []
        for i in range(len(c)):
            result.append(Dinv_diag[i] * c[i])
        
        # Clip negative
        for i in range(len(result)):
            if result[i] < 0.0: result[i] = 0.0
        
        # Normalize: result /= m^T @ result
        var m = self._compute_basis_integral(domain, Phi)
        var m_dot_r = 0.0
        for i in range(len(m)): m_dot_r += m[i] * result[i]
        if m_dot_r > 0.0:
            for i in range(len(result)): result[i] = result[i] / m_dot_r
        
        return result^
```

```mojo
def marginal_s_distribution(
    pdf: List[List[Float64]],
    v_weights: List[Float64],
) -> List[Float64]:
    var n_s = len(pdf)
    var marginal: List[Float64] = []
    for i in range(n_s):
        var val = 0.0
        for j in range(len(pdf[i])):
            val += pdf[i][j] * v_weights[j]
        marginal.append(val)
    return marginal^


def barrier_call_price(
    marginal: List[Float64],
    s_points_phys: List[Float64],
    s_weights: List[Float64],
    strike: Float64,
    discount: Float64,
    barrier: Float64,
) -> Float64:
    var price = 0.0
    for i in range(len(marginal)):
        if s_points_phys[i] <= barrier: continue
        var payoff = s_points_phys[i] - strike
        if payoff <= 0.0: continue
        price += marginal[i] * payoff * s_weights[i]
    return discount * price
```
